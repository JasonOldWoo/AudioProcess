
据网友测试：大多数手机上比较完美，没啥回声，但在小米上比较明显噪音。

这是一个开源项目
将实现各个平台上能快速使用的音频处理库
核心算法包括：
  NS（Noise Suppression 噪声抑制） 
  VAD（Voice Activity Detection 静音检测） 
  AECM（Acoustic Echo Canceller for Mobile 声学回声消除） 
  AGC（Auto Gain Control 自动增益控制）
现在只有一个Android Demo
